Configuring the Library

The SDHC driver library should be configured through the MHC. The following figures show the MHC configuration window for the SDHC driver and brief description.

Common User Configuration for all Instances
  • Driver Mode:
    • Allows User to select the mode of driver(Asynchronous or Synchronous).This setting is common for all the instances.
    • Only Asynchronous mode is supported for now
  • Number Of SDHC Instances:
    • Specifies number of SDHC Driver instances used. Currently maximum 1 instance is supported

 

Instance Specific User Configurations
  • Number Of SDHC Driver Clients:
    • Specifies number of clients to access the specific instance of the driver
  • File System For SDHC Driver Enabled:
    • Indicates whether SDHC driver will register its services with the file system or not.
    • This option is automatically checked when the file system is connected to the SDHC driver.
  • Buffer Queue Size:
    • Specifies maximum number of requests to be buffered in queue
    • When Connected to File system this value is set to 1 and non modifiable
  • Data Transfer Bus Width:
    • Specifies the Bus width to be used for data transfer (1-Bit, 4-Bit)
  • Maximum Bus Speed:
    • Specifies Bus Speed to be used for Communication with SD-card(DEFAULT_SPEED, HIGH_SPEED)
    • Standard SD-Cards only support Default Speed
    • SD-Cards with High Capacity support Both Default and High Speed
  • Use Write Protect (SDWP#) Pin:
    • When Selected Write protect check is enabled
    • A dedicated pin has to configured from Pin settings
  • Use Card Detect (SDCD#) Pin:
    • When Selected Dedicated pin can be used to detect the Card
    • A dedicated pin has to configured from Pin settings
  • DMA Channel for Transmit and Receive:
    • Specifies the DMA Channel configured for Transmit and Receive
  • RTOS Settings:
    • Stack Size:
      • Maximum Stack size to be allocated for SDHC Driver
    • Task Priority:
      • Specifies Priority for the SDHC driver task thread
    • Task Delay:
      • Specifies Delay to sleep for the SDHC task Thread after every call
